HTML Tag Reference Guide

This page demonstrates common HTML tags with explanations. Tags are elements that define how content is displayed.

Quick Nav: Text Formatting | Structure | Lists | Media | Forms

1. Text Formatting & Typography

Tags used to format text content.

Headings & Paragraphs

<h1> Largest Heading

<h6> Smallest Heading

<p> This is a standard paragraph used for blocks of text.

Inline Styling Tags

<b> Bold text (Stylistic)
<strong> Strong text (Important semantic meaning)
<i> Italic text (Stylistic)
<em> Emphasized text (Semantic emphasis)
<mark> Highlighted/Marked text
<small> Smaller text
This is <sub> subscript and this is <sup> superscript.
<del> Deleted/Strikethrough text
<ins> Inserted/Underlined text

Quotations & Code

<blockquote> This is a blockquote. It is usually indented and used for long quotations from another source.

Short quote: <q> This is a short inline quote.

Computer code:

<pre>
  The pre tag preserves   spaces
      and line breaks.
</pre>

Inline code: <code> const x = 10; </code>

Abbreviation: <abbr> HTML (Hover me)

<address> Written by Master Jayan.

2. Structural & Semantic Tags

Tags that define the layout and meaning of the page structure.

<div>: A non-semantic block container. Used purely for grouping.

<span>: A non-semantic inline container. Used for styling parts of text.

<article>

Represents a self-contained composition (like a blog post).

<aside>

Content indirectly related to the main content (like a sidebar).

<summary> Click to expand <details>

This content is hidden until you click the summary.
